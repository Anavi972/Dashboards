# COVID-19 Global Dashboard

This repository features an interactive **COVID-19 analytics dashboard** built using **Tableau**. The dashboard visualizes global pandemic trends, including confirmed cases, deaths, recoveries, and vaccinations, enabling users to explore COVID-19 data through intuitive and interactive visualizations.

---

## 📈 Project Overview

The dashboard provides a centralized view of worldwide COVID-19 statistics, allowing users to monitor infection trends, compare countries, analyze mortality rates, and track vaccination progress over time. It transforms large public health datasets into meaningful insights through interactive dashboards.

---

## 🖼️ Dashboard Showcase

The dashboard presents key pandemic metrics along with geographical and time-series visualizations, making it easy to explore global COVID-19 trends.

---

## ✨ Key Features & Analysis

### Executive KPIs

At-a-glance cards display important pandemic metrics:

- Total Confirmed Cases
- Total Deaths
- Total Vaccinations
- Death Rate
- Number of Countries Covered

### Global COVID Map

An interactive world map visualizes the distribution of confirmed COVID-19 cases across different countries and regions.

### Trend Analysis

Interactive line charts track:

- Daily New Cases
- Daily Deaths
- Cumulative Cases
- Vaccination Progress

### Country-wise Comparison

Compare countries based on:

- Total Cases
- Total Deaths
- Death Rate
- Vaccination Coverage

### Interactive Filters

The dashboard includes filters for:

- Country
- Continent
- Date

allowing users to perform customized analysis.

---

## 📊 Data Source

The dashboard uses publicly available COVID-19 datasets from **Our World in Data (OWID)** containing information such as:

- Country
- Date
- Population
- Total Cases
- New Cases
- Total Deaths
- Total Vaccinations
- New Vaccinations
